Cities: Skylines II Game Mechanics

How the simulation actually works

Plain-English explanations of CS2's game systems — written for players who want to understand what's really happening under the hood. Each topic covers the rules the game follows, common failure modes, and what you can do about them.

Content is derived from decompiled game source code, so the rules described here are the actual rules, not guesses from observed behaviour.
